# 成员管理与分享链接功能使用指南

## 📋 功能概述

SmartTable 现已支持完整的成员管理和分享链接功能，让您可以安全、便捷地与团队成员协作管理多维表格。

## 🎯 核心功能

### 1. 成员管理
- 支持 5 种角色权限：所有者 (Owner)、管理员 (Admin)、编辑者 (Editor)、评论者 (Commenter)、查看者 (Viewer)
- 添加/移除成员
- 动态调整成员角色
- 查看成员列表和权限

### 2. 分享链接
- 生成唯一的分享链接
- 支持"可编辑"和"仅查看"两种权限
- 可设置有效期（永久/7 天/30 天/自定义）
- 访问次数统计
- 一键复制链接

### 3. 首页菜单
- "我分享的"：查看您创建的所有分享
- "分享给我的"：查看他人分享给您的 Base

## 🚀 使用指南

### 成员管理

#### 打开成员管理对话框
1. 进入任意 Base 页面
2. 点击顶部工具栏的"成员"按钮
3. 打开成员管理对话框

#### 添加成员
1. 在成员管理对话框中，点击"添加成员"按钮
2. 输入用户邮箱
3. 选择角色权限：
   - **管理员**：管理 Base 设置和成员，不能删除 Base
   - **编辑者**：创建、编辑、删除记录和视图
   - **评论者**：查看数据，添加评论
   - **查看者**：仅查看数据
4. 点击"添加"按钮

#### 修改成员角色
1. 在成员列表中找到目标成员
2. 在角色下拉框中选择新角色
3. 角色立即生效

#### 移除成员
1. 在成员列表中找到目标成员
2. 点击"移除"按钮
3. 确认移除操作
4. 该成员将失去对所有 Base 的访问权限

### 分享链接

#### 创建分享链接
1. 进入 Base 页面
2. 点击"分享"按钮
3. 在分享对话框中：
   - 选择权限：可编辑 / 仅查看
   - 选择有效期：永久有效 / 7 天 / 30 天 / 自定义
4. 点击"创建分享链接"

#### 复制分享链接
1. 在分享列表中找到目标分享
2. 点击"复制链接"按钮
3. 链接已复制到剪贴板
4. 将链接发送给需要分享的用户

#### 管理分享链接
- **查看访问次数**：分享列表中显示每个链接的访问次数
- **删除分享**：点击"删除"按钮，确认后链接失效
- **过期时间**：自定义有效期时，到期后链接自动失效

### 首页菜单

#### 我分享的
1. 在首页点击"我分享的"菜单
2. 查看您创建的所有分享记录
3. 显示信息：
   - Base 名称和图标
   - 分享权限（可编辑/仅查看）
   - 访问次数
   - 创建时间
4. 点击记录可跳转到对应 Base

#### 分享给我的
1. 在首页点击"分享给我的"菜单
2. 查看其他用户分享给您的所有 Base
3. 显示信息：
   - Base 名称和描述
   - 分享者信息
   - 您的权限级别
4. 点击 Base 卡片可跳转并自动获得对应权限

| 编辑记录 | ✅ | ✅ | ✅ | ❌ | ❌ |

### 分享链接权限

- **可编辑**：通过链接访问的用户可以编辑 Base 内容
- **仅查看**：通过链接访问的用户只能查看，不能修改

## 🔒 安全特性

1. **唯一令牌**：每个分享链接使用 UUID 作为令牌，保证唯一性和安全性
2. **权限验证**：所有敏感操作前都会验证用户权限
3. **过期时间**：支持设置分享链接的过期时间
4. **访问控制**：防止越权访问，用户只能访问授权的 Base
5. **操作确认**：删除等敏感操作需要二次确认

## 💡 最佳实践

### 成员管理
- 只给必要的用户授予管理员权限
- 定期审查成员列表，移除不再需要访问的用户
- 为临时协作者设置较短的有效期

### 分享链接

- 设置合理的有效期，避免长期有效

- 监控访问次数，发现异常及时处理

## 📊 API 接口

### 成员管理 API

```bash
# 获取成员列表
GET /api/bases/{base_id}/members

# 添加成员
POST /api/bases/{base_id}/members
{
  "email": "user@example.com",
  "role": "editor"
}

# 更新成员角色
PUT /api/bases/{base_id}/members/{user_id}
{
  "role": "admin"
}

# 移除成员
DELETE /api/bases/{base_id}/members/{user_id}
```

### 分享链接 API

```bash
# 创建分享链接
POST /api/bases/{base_id}/shares
{
  "permission": "edit",
  "expires_at": 1712476800
}

# 获取分享列表
GET /api/bases/{base_id}/shares

# 删除分享链接
DELETE /api/shares/{share_id}

# 访问分享链接
GET /api/share/{share_token}

# 获取我创建的分享
GET /api/bases/shared-by-me

# 获取分享给我的 Base
GET /api/bases/shared-with-me
```

## ❓ 常见问题

### Q: 如何撤销对某个用户的分享？
A: 在分享列表中找到对应的分享链接，点击"删除"按钮即可。

### Q: 分享链接过期后还能续期吗？
A: 目前不支持续期，需要重新创建分享链接。

### Q: 成员被移除后还能看到之前的数据吗？
A: 不能，成员被移除后立即失去所有访问权限。

### Q: 如何知道谁通过我的分享链接访问了 Base？
A: 在分享列表中可以查看每个链接的访问次数。

### Q: 可以限制分享链接的访问次数吗？
A: 目前不支持访问次数限制，但可以设置过期时间。
