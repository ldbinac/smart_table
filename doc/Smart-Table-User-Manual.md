## 1. 项目概述

**Smart Table** 是一款功能强大的在线智能数据表格管理系统，类似于 Airtable / 飞书多维表格，旨在让数据管理更简单、更高效。系统支持多种视图模式、丰富的字段类型、数据分组筛选排序、仪表盘可视化、表单收集、协作分享等核心能力，适用于项目管理、客户管理、任务跟踪、库存管理等多种业务场景。

### 核心特性一览

| 特性 | 说明 |
|------|------|
| 多维表管理 | 创建、编辑、删除、收藏多维表格，支持自定义图标和颜色 |
| **26 种字段类型** | 文本、数字、日期、日期时间、单选/多选、复选框、附件、关联、公式、自动编号等 |
| 6 种视图模式 | 表格、看板、日历、甘特图、画廊、表单 |
| 数据分组 | 最多 3 级分组，分组内独立编号 |
| 筛选与排序 | 多条件组合筛选（AND/OR）、多字段排序，配置持久化 |
| **公式引擎** | **43 个内置函数**，支持数学、文本、日期、逻辑、统计计算 |
| 数据导入/导出 | 支持 CSV、Excel、JSON 格式，支持 Excel 导入创建数据表 ⭐ |
| 仪表盘 | 可视化数据展示，支持多种组件和**仪表盘模板** ⭐ |
| 协作与分享 | 成员管理、链接分享、权限控制（Base/表单/仪表盘三种分享方式） |
| 实时协作 | 多人在线编辑、在线状态、单元格锁定、冲突解决、离线队列 ⭐ |
| 表单收集 | 自定义表单、公开分享、收集填写人信息 |
| 模板系统 | 10 种预置模板快速创建 |
| **Element Plus 图标** | 统一的视觉风格，提升用户体验 ⭐ |
| **API 文档系统** | 完整的 Swagger/OpenAPI 文档 ⭐ |
| 操作历史 | 记录所有数据变更，支持历史追溯 |
| **邮件系统** | SMTP 邮件发送、模板管理、异步队列（可选）⭐ |
| **安全防护** | XSS 防护、速率限制、CSRF 保护、安全响应头 ⭐ |

> ⭐ 表示 v1.2.0 新增或增强的功能

## 3. 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9
- Python >= 3.9 （仅后端模式需要）

### 一键启动（推荐）

```bash
# Windows PowerShell
.\start.ps1

# Linux/macOS
./start.sh
```

脚本会自动：
1. 安装前端依赖并启动开发服务器（端口 5173）
2. 安装后端依赖并启动 Flask 服务器（端口 5000）
3. 打开浏览器访问 http://localhost:5173

### 方式一：Docker 部署

```bash
# 克隆仓库
git clone <repository-url>
cd smart-table-spec

# 复制环境变量配置
cp .env.example .env
# 编辑 .env 文件配置数据库连接等（默认使用 SQLite）

# 使用 Docker Compose 启动所有服务
docker-compose up -d

# 或使用 PostgreSQL + Redis（适合生产环境）
docker-compose -f docker-compose.dev.yml up -d

# 执行数据库迁移
docker-compose exec backend flask db upgrade

# 访问系统
# http://localhost (前端)
# http://localhost:5000/api/docs (API 文档 - Swagger UI)
# http://localhost:5000/api/apidoc (API 文档 - ReDoc)
```

### 方式二：手动部署

```bash
# 启动后端（可选）
cd smarttable-backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
cp .env.example .env
flask db upgrade
flask run --reload
# 或启用实时协作: python run.py --enable-realtime

# 启动前端
cd ../smart-table
npm install
npm run dev
```

#### 前端开发常用命令

```bash
npm run dev        # 开发模式
npm run build      # 构建生产版本
npm run preview    # 预览构建结果
npm run test       # 运行测试
npm run test:watch # 监听模式运行测试
npm run test:coverage # 生成测试覆盖率报告
```

#### 后端特性说明

✅ **默认数据库**: SQLite（轻量级，无需额外安装）  
✅ **可选数据库**: PostgreSQL（通过环境变量 `DATABASE_URL` 配置）  
✅ **认证系统**: JWT Token 认证，支持刷新 Token、邮箱验证、密码重置  
✅ **权限管理**: 基于角色的访问控制（RBAC）  
✅ **数据库迁移**: Alembic 迁移工具（已包含 8 个迁移版本）  
✅ **API 文档**: 完整的 Swagger/OpenAPI 文档（Flasgger）⭐  
✅ **实时协作**: 可选的 WebSocket 实时协作功能  
✅ **邮件系统**: 可选的 SMTP 邮件发送功能 ⭐  
✅ **对象存储**: 可选的 MinIO 文件存储 ⭐  
✅ **安全防护**: XSS 防护、速率限制、安全响应头 ⭐

## 4. 功能模块说明

### 4.1 用户认证

#### 注册

1. 访问系统，点击「注册」进入注册页面
2. 填写注册信息：用户名（3-20字符）、邮箱、密码（≥8位，含大小写和数字）、验证码
3. 点击「注册」按钮完成注册
4. 系统会发送邮箱验证邮件（如果启用了邮件服务）
5. 注册成功后自动跳转到登录页面

#### 登录

1. 输入邮箱、密码和验证码
2. 点击「登录」按钮
3. 登录成功后跳转到首页
4. 系统使用 JWT Token 进行身份验证，支持 Token 自动刷新

> ⚠️ **安全提示**：连续5次登录失败后账户将被锁定15分钟

#### 忘记密码

1. 在登录页面点击「忘记密码」
2. 输入邮箱和验证码
3. 系统发送密码重置邮件
4. 点击邮件中的重置链接设置新密码

#### 修改密码

1. 登录系统后，点击右上角用户头像
2. 选择「设置」
3. 在设置页面中找到「修改密码」
4. 输入当前密码和新密码
5. 点击「确认修改」

#### 退出登录

1. 点击右上角用户头像 → 选择「退出登录」
2. 系统清除本地 Token 并跳转到登录页

### 4.2 首页与多维表管理

首页提供四个导航标签：首页（收藏+全部卡片）、全部（列表+分页）、模板（10种预置模板）、分享（我分享的/分享给我的）

#### 创建多维表

1. 点击首页右上角「新建」按钮
2. 在弹出对话框中选择创建方式：
   - **创建空白多维表**：从零开始创建
   - **从模板创建**：选择预置模板快速创建
3. 填写多维表信息：
   - **名称**（必填，1-50字符）
   - **描述**（选填，最多200字符）
   - **图标**：从10个预设图标中选择
   - **颜色**：从8种预设颜色中选择
4. 点击「创建」完成

#### 管理多维表

- **编辑**：点击卡片右下角菜单 → 编辑，修改名称、描述、图标和颜色
- **删除**：点击卡片右下角菜单 → 删除（删除操作不可恢复）
- **收藏/取消收藏**：点击卡片上的星标图标
- **搜索**：在顶部搜索栏输入关键词搜索多维表
- **打开**：点击卡片直接打开多维表（在新标签页中打开）

#### 分享视图

在「分享」标签页中：

- **我分享的**：查看自己创建的分享链接，支持复制链接和删除分享
- **分享给我的**：查看其他用户分享给我的多维表

### 4.3 数据表操作

进入多维表后，左侧边栏显示数据表和仪表盘列表。

#### 创建数据表

**方式一**：点击「+ 新建数据表」→ 输入名称和描述 → 确定（自动创建默认字段和表格视图）

**方式二**：选择「从 Excel 导入创建」⭐ → 上传 Excel 文件 → 系统智能解析（识别字段类型、提取选项值）→ 配置字段 → 创建并导入数据

#### 管理数据表

选择/重命名/复制/删除/拖拽排序/批量管理

### 4.4 字段管理

支持 **26 种字段类型**，分为文本(3种)⭐、数值(1种)、日期(2种)、选择(3种)、人员(1种)、联系(3种)、媒体(1种)、计算(1种)、关联(1种)、查找(1种)、系统(5种)⭐、其他(2种)

#### 添加/管理字段

添加字段 → 配置名称/类型/必填/默认值/特有选项 → 确定
编辑/删除/排序/隐藏显示/冻结列/调整列宽/批量排序

---

### 4.5 视图管理

支持 **6 种视图类型**：表格、看板、日历、甘特图、画廊、表单

每个视图独立保存筛选、排序、分组、隐藏/冻结字段、行高、列宽等配置

---

### 4.6 记录管理

支持多种添加方式（行内/抽屉/视图/分组/表单提交）、行内/抽屉编辑、单条/批量删除、复制记录、查看详情和历史记录

**保存优化** ⭐：保存后自动关闭抽屉，创建后强制刷新避免重复

---

### 4.7 筛选功能

支持 **20+ 种操作符**：基础比较(等于/不等于/为空/不为空)、文本匹配(包含/不包含/开头是/结尾是/是以下之一等)、数值比较(大于/小于等)、日期比较(在范围内/早于/晚于/今天等)、布尔判断、存在性

支持多条件 AND/OR 组合、通配符、正则表达式（高级）、保存筛选方案

---

### 4.8 排序功能

多级排序（拖拽调整优先级）、空值处理、特殊字段排序规则、单击列头快捷排序

---

### 4.9 分组功能

最多 **3 级分组**，支持展开/折叠、独立编号、快速添加、表头固定、样式定制、分组统计

支持分组的字段类型：单选、复选框、日期、成员、创建时间、关联

---

### 4.10 公式引擎 ⭐

支持 **43 个内置函数**：

- **数学函数(11个)**：SUM, AVG, MAX, MIN, ROUND, CEILING, FLOOR, ABS, MOD, POWER, SQRT
- **文本函数(10个)**：CONCAT, LEFT, RIGHT, LEN, UPPER, LOWER, TRIM, SUBSTITUTE, REPLACE, FIND
- **日期函数(10个)**：TODAY, NOW, YEAR, MONTH, DAY, HOUR, MINUTE, SECOND, DATEDIF, DATEADD
- **逻辑函数(7个)**：IF, AND, OR, NOT, IFERROR, IFS, SWITCH
- **统计函数(5个)**：COUNT, COUNTA, COUNTIF, SUMIF, AVERAGEIF

语法示例：`{单价} * {数量}`、`IF({成绩}>=90,"优秀",IF({成绩}>=60,"及格","不及格"))`

### 4.11 数据导入与导出

#### 导入到现有表

上传 CSV/Excel/JSON → 预览 → 配置字段映射和选项 → 开始导入（显示进度）→ 自动刷新

#### Excel 导入创建表 ⭐

上传 Excel → 智能解析（识别26种字段类型、提取选项、处理多选逗号分隔）→ 配置字段和主字段 → 显示进度（创建结构→导入数据→完成）→ 自动打开

智能识别：文本/长文本/富文本⭐、数字、日期/日期时间⭐、单选/多选/复选框、邮箱/电话/URL

#### 导出数据

选择格式(CSV/Excel/JSON)、范围(全部/选中/当前页)、字段(可见/自定义)、其他选项 → 下载

兼容性改进 ⭐：字段类型映射优化、大数据量支持、错误恢复、编码处理、进度显示

### 4.13 分享与协作

#### 三层分享机制

1. **Base 分享**：创建链接 → 设置权限(查看/编辑)/过期时间/密码/访问次数 → 管理(查看统计/编辑/删除/重新生成)
2. **表单分享**：公开表单 → 配置提交选项(允许重复/收集填写人信息) → 分发链接 → 数据自动写入
3. **仪表盘分享**：公开仪表盘 → 实时数据展示 → 支持嵌入外部网站

#### 成员管理

查看列表/添加成员(邮箱邀请/搜索/批量)/修改角色/移除成员/搜索/统计

#### 权限角色

Owner(完全控制) > Admin(管理除删除外) > Editor(编辑) > Commenter(评论) > Viewer(只读)

---

### 4.14 实时协作 ⭐

> 需后端启用 `--enable-realtime` 或 `ENABLE_REALTIME=true`，未启用则自动隐藏相关UI

#### 核心功能

- **在线状态**：AppHeader 显示在线用户头像列表（全局显示）⭐
- **单元格锁定**：编辑时自动锁定，显示编辑者，超时自动释放
- **冲突解决**：乐观锁检测 → 弹出对话框 → 覆盖/放弃/合并
- **离线队列**：断线时缓存操作(IndexedDB,最多100条) → 重连自动重放(FIFO)
- **优雅降级**：WebSocket不可用时降级为普通HTTP模式
- **连接状态**：🟢已连接/🟡重连中/🔴已断开/⚪未启用

#### Socket.IO事件

房间管理(room:join/leave)、在线状态(presence:*)、单元格锁定(lock:*)、数据同步(data:*)

---

### 4.15 表单功能

#### 创建和配置

创建表单视图 → 配置(标题/描述/提交按钮/成功提示/重复提交/字段可见性/外观) → 公开分享(无需登录) → 收集数据自动写入

#### 提交记录

在数据表中查看，支持按时间筛选和导出

---

### 4.16 系统设置

外观(主题/语言/图标)、表格(行高/网格线/斑马纹/记录数)、数据(自动保存/离线模式)、格式(日期/时间/数字/货币/时区)、通知(桌面/邮件/协作)、重置默认

## 5. 字段类型详解

### 5.1 文本类型（3种）⭐

| 类型 | 说明 | 配置 | 示例 |
|------|------|------|------|
| 单行文本 | 短文本输入 | 最大长度1-500 | 姓名、标题 |
| 长文本 ⭐ | 多行段落输入 | 最大长度1-10000 | 描述、备注 |
| 富文本 ⭐ | HTML编辑器 | HTML标签白名单、XSS防护⭐ | 文章、详情 |

### 5.2 数值类型（1种）

**数字**：精度0-10位小数、格式(数字/货币/百分比)、符号、前缀后缀、千分位

### 5.3 日期类型（2种）

- **日期**：日期选择器，多种格式
- **日期时间 ⭐**：日期+时间，精确到秒，时区设置

### 5.4 选择类型（3种）

- **单选**：选项列表(名称+颜色≤100个)、允许新增
- **多选**：同上，最大选择数
- **复选框**：布尔开关，默认值，显示样式

### 5.5 人员类型（1种）

**成员**：用户选择，默认值(无/当前用户)⭐，MemberSelect组件⭐(搜索/头像/最近使用)

### 5.6 联系方式（3种）

电话(格式/拨打)、邮箱(验证/发邮件)、URL(favicon/新窗口)

### 5.7 媒体类型（1种）

**附件**：MIME白名单、大小限制(默认10MB)、数量限制(0-100)、缩略图⭐

增强功能 ⭐：
- 后端上传(MinIO对象存储)、前端代理访问
- 图片缩略图(小/中/大)、MIME二次验证
- 批量上传/拖拽上传/进度条/预览(图片/PDF/视频/音频)

### 5.8 计算类型（1种）

**公式**：只读计算字段，43函数，精度设置，详见[4.10](#410-公式引擎)

### 5.9 关联类型（1种）

**Link**：目标表、关系类型(一对一/一对多/多对一/多对多)、双向关联、多选

### 5.10 查找类型（1种）

**Lookup**：关联字段+目标字段+聚合类型(First/Last/Concat/Sum/Avg/Min/Max/Count)

### 5.11 系统类型（5种）

Created By/ Created Time/ Updated By/ Updated Time (自动填充)

**Auto Number ⭐** (新)：前缀/后缀/日期前缀(YYYYMMDD等)/补零位数(如4位→0001)/起始编号

示例：前缀"TASK-"+YYYYMMDD+4位补零 → TASK-202604260001

### 5.12 其他类型（2种）

**Rating**：1-10星、半星、样式(星/数字/进度条)
**Progress**：进度条/圆形/百分比、颜色阈值

## 10. 常见问题解答（FAQ）

### 基础问题

**Q1: 如何恢复误删？**
A: 目前不可恢复。建议开启删除确认、定期导出备份、操作前复制

**Q2: 数据存在哪？**
A: 后端模式存PostgreSQL/SQLite；纯前端存IndexedDB；混合模式优先API后降级缓存

**Q3: 支持离线？**
A: 部分支持。IndexedDB缓存读取，写操作需后端，离线队列缓存操作待重放

**Q4: 如何改API地址？**
A: 修改 `.env.development` 的 `VITE_API_BASE_URL`，重启dev server

**Q5: 筛选排序会保存吗？**
A: 会。持久化到数据库/IndexDB，切换视图/刷新不丢，每视图独立

**Q6: 分组最多几级？**
A: 3级。每级独立展折，树形层级

**Q7: 公式支持哪些函数？**
A: 43个，详见[4.10](#410-公式引擎)。数学11+文本10+日期10+逻辑7+统计5

**Q8: 附件限制？**
A: MIME白名单、单文件≤10MB(可调≤100MB)、总数限制、0-100个、缩略图、服务端二次验证

**Q9: 批量操作？**
A: 表格勾选多行→右键→批量删除(未来扩展更多)

**Q10: 看板如何移动？**
A: 拖拽卡片到目标列，自动更新分组字段值

**Q11: 自定义表单？**
A: 配置对话框中设标题/描述/按钮文字/成功提示/可见字段顺序/字段说明/宽度对齐/Logo

**Q12: 仪表盘实时更新？**
A: 支持。组件设置中开启自动刷新(10秒~30分钟间隔)或手动刷新

### 账户安全

**Q13: 忘记密码？**
A: 登录页→忘记密码→输邮箱验证码→收到重置邮件→点链接→设新密码(24h有效)
> v1.2.0已修复验证绕过漏洞⭐

**Q14: 账户锁定？**
A: 连续5次失败→锁15分钟→到期解锁或管理员手动解锁→记操作日志含IP

**Q15: 收不到验证邮件？**
A: 检查垃圾箱/未配SMTP/邮箱错/延迟/防火墙拦截→设置中重发或联系管理员加白名单

**Q16: 密码要求？**
A: ≥8位，必须大写+小写+数字，建议加特殊字符。例：MyP@ssw0rd2026!

**Q17: 如何启用实时协作？**
A: `python run.py --enable-realtime` 或 `ENABLE_REALTIME=true` 或 Docker加环境变量
未启用则前端自动隐藏协作UI，其他功能正常

**Q18: 看不到在线用户？**
排查：1.后端未启用(最常见) 2.连接状态(底部指示器) 3.无其他人打开同表 4.浏览器兼容(Chrome90+) 5.广告拦截插件 6.代理VPN阻断WS

**Q19: 冲突怎么处理？**
乐观锁检测→弹窗三选项：覆盖(你的覆盖他的，慎用)、放弃(加载最新)、合并(不同字段才可)
预防：先看是否有人在编辑、及时保存减少窗口期

### 功能使用

**Q20: Excel导入支持哪些类型？**
A: 智能识别文本(单行/长/富⭐)、数字、日期/日期时间⭐、选择(单/多/复选)、联系(邮/电/URL)、系统(自动编号⭐)
不支持：公式/关联/查找/附件/成员/系统字段(需手动)

**Q21: 多选选项如何提取？**
遍历列→按`,/，/;/\|`分割→trim→去重→限20个→按频排序→分配20色
例："A,B,A"+"C,D"B" → ["A"(3),"B"(2),"C"(1),"D"(1)]

**Q22: 如何设主字段？**
字段配置步骤中点字段行左侧○按钮→建议首列→作用：第一列/关联显示/搜索标题/导出标识

**Q23: 导入能取消吗？**
v1.2.0为原子操作不建议中途取消。中断可能产生不完整表(可手动删)。建议先确
