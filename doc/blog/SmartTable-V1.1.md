# SmartTable 1.1 来了：你的团队协作，从此多了一种可能

---

如果你用过 SmartTable 1.0，你大概已经习惯了它丝滑的表格操作、灵活的视图切换和强大的公式引擎。但你可能也曾在某个瞬间想过——

**"要是能和同事一起编辑就好了。"**

**"要是能直接发邮件通知就好了。"**

**"要是数据安全再稳一点就好了。"**

现在，这些"要是"都变成了"已经"。

SmartTable 1.1 正式发布。这不是一次简单的版本迭代，而是一次从"个人工具"到"团队平台"的跨越。

---

## 先说最重要的：你们终于可以一起用了

### 实时协作，所见即所得

打开同一个多维表格，你能看到谁在线、谁在编辑哪个单元格。别人改了数据，你的屏幕上几乎是同时更新。没有"刷新一下看看最新版"的尴尬，没有"你先改完我再改"的等待。

更贴心的是，当有人正在编辑某个单元格时，这个单元格会自动锁定——你不会不小心覆盖掉同事刚输入的内容。如果真的碰上了同时修改同一条记录的情况，系统会智能检测冲突，弹出提示让你选择保留哪个版本。

这一切的背后是 WebSocket 实时通信 + Redis 状态管理的技术支撑，但对使用者来说，感受只有一个字：**顺**。

> **场景**：产品团队在同一个需求表里协作，产品经理写需求描述，设计师更新设计状态，开发标注技术方案——所有人同时在线，数据实时同步，再也不用群里喊"我改完了你看看"。

## 邮件服务：该通知的，一个都不落

### 从手动通知到自动触达

以前新增一个团队成员，你得手动发消息告诉他账号创建好了。用户忘记密码，只能找管理员后台重置。现在，这些事情系统自己就能搞定。

1.1 版本内置了完整的邮件服务系统：

- **新用户注册**，自动发送欢迎邮件和邮箱验证链接
- **忘记密码**，用户自助点击"忘记密码"，系统秒发重置链接
- **密码修改**，及时邮件通知，确保账户安全

管理员还能在后台自定义邮件模板，调整样式和文案，让每一封系统邮件都带着你们团队的专属风格。邮件发送支持队列和自动重试，即使 SMTP 服务偶尔抽风，也不会丢邮件。

> **场景**：新同事入职，注册账号后立刻收到欢迎邮件和验证链接，三分钟完成账号激活。没人需要额外花时间做这些事。

## 安全：看不见的底气

### 不只是"够用"，而是"放心"

数据安全这件事，平时感觉不到，出了问题就是大问题。1.1 版本在安全层面做了全方位的加固，你可能不会注意到这些变化，但它们一直在默默守护你的数据。

**XSS 防护**：所有用户输入的 HTML 内容，在渲染前都会经过 DOMPurify 消毒处理。恶意脚本？进不来。

**速率限制**：登录接口连续尝试5次就会被锁定15分钟。暴力破解？没门。

**文件上传安全**：不光看文件扩展名，还会读取文件头部的"Magic Number"来验证真实类型。把 `.exe` 改成 `.png` 想混进来？不可能。

**安全响应头**：X-Content-Type-Options、X-Frame-Options、HSTS……这些专业级安全头全部配齐，防点击劫持、防 MIME 嗅探、强制 HTTPS，一个不少。

**异常信息隔离**：API 出错时，返回给用户的是友好的通用提示，内部错误详情只记录在服务端日志里。数据库表名、文件路径这些敏感信息，永远不会暴露给前端。

> **场景**：你的多维表格里存着客户数据和财务信息。1.1 的安全体系让你不用担心有人通过 XSS 窃取数据，不用担心暴力破解撞库，也不用担心系统报错时泄露内部信息。

## 仪表盘：数据会说话

### 从数字到洞察，一步之遥

数据躺在表格里是一回事，能一眼看出趋势是另一回事。1.1 版本的仪表盘功能，让你的数据真正"活"起来。

拖拽式布局，想放哪里放哪里。柱状图看对比，折线图看趋势，饼图看占比，KPI 卡片看核心指标。还有实时时钟、日期组件、跑马灯这些实用小部件，让仪表盘既专业又生动。

做好了仪表盘？一键生成分享链接，设置密码和过期时间，发给老板、客户或者合作伙伴，他们无需登录就能看到你精心准备的数据看板。

> **场景**：每周例会，你不再需要花半小时做 PPT。仪表盘实时展示本周数据，销售趋势、客户分布、项目进度一目了然。分享链接发到群里，所有人随时查看最新数据。

## 管理员面板：掌控全局

### 用户管理、系统配置、操作日志，一站搞定

1.1 为管理员准备了完整的后台管理面板：

- **用户管理**：查看所有用户，按角色和状态筛选，批量操作，一键重置密码
- **系统配置**：邮件 SMTP 设置、系统参数调整，不用改代码
- **操作日志**：谁在什么时候做了什么操作，清清楚楚，支持 CSV 导出
- **邮件管理**：模板编辑、发送日志、统计报表，邮件系统尽在掌握

> **场景**：有用户反馈收不到验证邮件，管理员打开后台一看，发送日志里明明白白记录着失败原因和重试状态，三分钟定位问题。

## Docker 一键部署：十分钟上线

### 从开发环境到生产环境，不再是噩梦

1.1 版本提供了完整的 Docker 部署方案。一条命令，前端构建、后端部署、Nginx 配置、进程管理，全部自动完成。

```bash
git clone https://github.com/ldbinac/smart_table.git
cd smart_table
docker-compose up -d
```

就这样，系统跑起来了。

需要 PostgreSQL + Redis 的完整版？`docker-compose.full.yml` 已经准备好了。想自己调整 Nginx 配置？配置文件都在 `docker/` 目录下，清清楚楚。

## 还有一些你可能没注意到的细节

- **内存泄漏修复**：多个组件的事件监听器在卸载时正确清理，长时间使用不再越来越卡
- **生产日志安全**：生产环境自动移除所有 `console.log`，敏感信息不会意外泄露到浏览器控制台
- **密码强度验证**：注册和重置密码时强制要求大写字母 + 小写字母 + 数字，最低8位
- **验证码防护**：登录、注册、表单提交都支持验证码，防机器人防刷量
- **多维表复制**：一键复制整个多维表，快速创建相似结构的新项目
