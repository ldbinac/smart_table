#### 📈 仪表盘模板系统 ⭐

全新的一键复用仪表盘配置的功能：

**保存为模板**：

- 将精心设计的仪表盘布局和组件配置保存为可复用的模板
- 可选择是否包含数据源绑定（建议不包含，便于跨表复用）
- 设置模板名称、描述、分类标签
- 个人模板仅自己可见


**管理我的模板**：

- 查看、编辑、删除个人保存的模板
- 分享模板给团队成员（需管理员权限）
- 模板版本管理（更新模板后保留历史版本）

**从模板创建**：

- 浏览**模板库**（系统预设行业模板 + 个人保存的模板）
- 预览模板效果（查看布局和组件配置）

**MinIO 对象存储支持**：

- 可选集成 MinIO 作为文件对象存储（替代本地文件系统）
- 支持分布式部署和多节点共享存储
- 自动创建 Bucket 和目录结构
- 大文件分片上传支持

**前端代理访问**：

- 通过 API 代理下载文件（隐藏真实存储路径）
- 支持 Range 请求（视频/大文件断点续传）
- 访问权限校验（防止未授权访问私有文件）
- CDN 缓存友好（支持 Cache-Control 头）
病毒扫描接口（可选集成 ClamAV）
