# 首页多维表格管理功能完善计划

## 问题概述

首页（Home.vue）目前只实现了创建多维表格和显示列表的功能，但缺少对多维表格的管理操作：
1. **重命名** - 无法修改已创建的多维表格名称
2. **删除** - 无法删除不需要的多维表格
3. **收藏** - 无法标记常用的多维表格

## 修复步骤

### 步骤1: 在首页添加操作菜单

**文件**: `src/views/Home.vue`

**修改内容**:
- 在每个多维表格卡片上添加操作按钮（更多选项）
- 操作菜单包含：重命名、收藏/取消收藏、删除
- 点击卡片主体仍然进入该多维表格

### 步骤2: 实现重命名功能

**文件**: `src/views/Home.vue`

**实现内容**:
1. 添加重命名对话框
2. 表单包含：名称、描述、图标、颜色
3. 调用 `baseStore.updateBase` 更新数据
4. 更新成功后刷新列表

### 步骤3: 实现删除功能

**文件**: `src/views/Home.vue`

**实现内容**:
1. 点击删除时显示确认对话框
2. 确认后调用 `baseStore.deleteBase` 删除
3. 从列表中移除已删除的多维表格
4. 显示成功/失败提示

### 步骤4: 实现收藏功能

**文件**: `src/views/Home.vue`

**实现内容**:
1. 在卡片上显示收藏状态（星标图标）
2. 点击收藏按钮切换收藏状态
3. 调用 `baseStore.toggleStarBase` 切换
4. 收藏的卡片显示星标标识
5. 列表按收藏状态排序（收藏的在前）

### 步骤5: 优化卡片样式

**文件**: `src/views/Home.vue`

**优化内容**:
1. 添加操作按钮的悬停效果
2. 收藏状态的视觉标识
3. 操作菜单的样式优化
4. 防止点击操作按钮时触发卡片点击事件

### 步骤6: 全面测试

**测试内容**:
1. 重命名功能：
   - 打开重命名对话框
   - 修改名称、描述、图标、颜色
   - 保存后列表更新

2. 删除功能：
   - 点击删除显示确认框
   - 确认后从列表移除
   - 取消后保持原样

3. 收藏功能：
   - 点击收藏按钮切换状态
   - 星标图标正确显示
   - 列表按收藏状态排序

4. 边界情况：
   - 空名称验证
   - 删除最后一个多维表格
   - 快速连续操作
