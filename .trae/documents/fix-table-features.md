# 表格功能修复计划

## 问题概述

1. **按钮功能缺失**: "字段"、"筛选"、"排序"和"导出"四个按钮目前只是占位符，没有实际功能
2. **输入框字体颜色问题**: 添加记录后，数据行输入内容字体颜色过浅，无法清晰查看

## 修复步骤

### 步骤1: 修复输入框字体颜色问题

**文件**: `src/components/views/TableView/TableCell.vue`

**问题分析**:
- 编辑状态下的 `.cell-input` 样式没有明确设置字体颜色
- 默认继承的颜色可能过浅

**修复方案**:
- 在 `.cell-input` 和 `.cell-select` 样式中添加明确的字体颜色 `color: $text-primary`

### 步骤2: 实现"字段"按钮功能

**文件**: `src/views/Base.vue`

**功能需求**:
- 点击"字段"按钮打开字段管理面板
- 面板功能包括：
  - 显示当前表格的所有字段
  - 添加新字段
  - 编辑字段属性（名称、类型、选项等）
  - 删除字段
  - 调整字段顺序
  - 显示/隐藏字段

**实现方案**:
1. 创建字段管理对话框组件或内联面板
2. 在 Base.vue 中添加字段按钮的点击处理函数
3. 集成 fieldService 进行字段的 CRUD 操作

### 步骤3: 实现"筛选"按钮功能

**文件**: `src/views/Base.vue`

**功能需求**:
- 点击"筛选"按钮打开筛选面板
- 面板功能包括：
  - 添加筛选条件（字段、操作符、值）
  - 多条件组合（AND/OR）
  - 保存/清除筛选条件
  - 实时显示当前筛选条件

**实现方案**:
1. 创建筛选面板组件（可以使用已有的 FilterPanel）
2. 在 Base.vue 中添加筛选按钮的点击处理
3. 集成 filter.ts 中的筛选逻辑
4. 将筛选条件应用到表格数据显示

### 步骤4: 实现"排序"按钮功能

**文件**: `src/views/Base.vue`

**功能需求**:
- 点击"排序"按钮打开排序面板
- 面板功能包括：
  - 添加排序字段
  - 选择排序方向（升序/降序）
  - 多字段排序优先级设置
  - 保存/清除排序条件

**实现方案**:
1. 创建排序面板组件（可以使用已有的 SortPanel）
2. 在 Base.vue 中添加排序按钮的点击处理
3. 集成 sort.ts 中的排序逻辑
4. 将排序条件应用到表格数据显示

### 步骤5: 实现"导出"按钮功能

**文件**: `src/views/Base.vue`

**功能需求**:
- 点击"导出"按钮打开导出选项对话框
- 导出选项包括：
  - 导出格式（Excel、CSV、JSON）
  - 导出范围（全部记录、选中记录、当前视图）
  - 导出字段选择

**实现方案**:
1. 创建导出对话框组件
2. 在 Base.vue 中添加导出按钮的点击处理
3. 集成已有的导出功能（export/index.ts）
4. 实现文件下载功能

### 步骤6: 全面测试

**测试内容**:
1. 输入框字体颜色是否正常显示
2. 字段按钮功能：
   - 打开字段管理面板
   - 添加/编辑/删除字段
   - 字段显示/隐藏
3. 筛选按钮功能：
   - 打开筛选面板
   - 添加/编辑/删除筛选条件
   - 筛选结果正确性
4. 排序按钮功能：
   - 打开排序面板
   - 添加/编辑/删除排序条件
   - 排序结果正确性
5. 导出按钮功能：
   - 打开导出对话框
   - 不同格式导出
   - 文件下载正常

## 依赖文件

- `src/views/Base.vue` - 主页面，包含四个按钮
- `src/components/views/TableView/TableCell.vue` - 单元格组件，需要修复字体颜色
- `src/utils/filter.ts` - 筛选逻辑
- `src/utils/sort.ts` - 排序逻辑
- `src/db/services/fieldService.ts` - 字段服务
- `src/export/index.ts` - 导出功能
- `src/stores/baseStore.ts` - 状态管理
- `src/stores/viewStore.ts` - 视图状态管理
