# 字段显示/隐藏功能实现计划

## 任务概述
在字段管理界面中实现字段显示/隐藏功能，支持持久化存储，并在所有视图和弹窗中正确应用隐藏状态。

## 实现步骤

### 阶段一：数据模型与存储层

#### 1.1 更新数据库 Schema
- **文件**: `src/db/schema.ts`
- **内容**: 在 `Field` 接口中添加 `isVisible` 字段（布尔类型，默认为 true）
- **影响**: 需要更新 Dexie 数据库版本

#### 1.2 更新字段服务层
- **文件**: `src/db/services/fieldService.ts`
- **内容**: 
  - 添加 `updateFieldVisibility(fieldId: string, isVisible: boolean)` 方法
  - 在创建字段时设置默认 `isVisible: true`

#### 1.3 更新类型定义
- **文件**: `src/types/fields.ts`
- **内容**: 在 `FieldOptions` 中添加 `isVisible` 属性

### 阶段二：字段管理界面

#### 2.1 更新字段管理对话框
- **文件**: `src/components/dialogs/FieldDialog.vue`
- **内容**:
  - 在字段列表中为每个字段添加显示/隐藏切换按钮（Switch 组件）
  - 添加字段可见性图标标识（眼睛图标显示/隐藏）
  - 实现切换时调用 `fieldService.updateFieldVisibility`
  - 添加提示文字说明功能

#### 2.2 添加字段可见性状态管理
- **文件**: `src/stores/tableStore.ts` 或 `src/stores/viewStore.ts`
- **内容**:
  - 添加 `hiddenFields` 状态管理
  - 提供 `toggleFieldVisibility(fieldId)` action
  - 提供 `isFieldVisible(fieldId)` getter

## 技术实现细节

### 数据流设计
```
用户点击切换按钮
    ↓
FieldDialog 调用 fieldService.updateFieldVisibility
    ↓
更新 IndexedDB 中的字段数据
    ↓
更新 Pinia store 中的 fields 状态
    ↓
Vue 响应式系统自动更新所有使用 fields 的组件
    ↓
所有视图和弹窗自动刷新
```

### 关键代码示例

#### 字段可见性切换
```vue
<el-switch
  v-model="field.isVisible"
  :active-value="true"
  :inactive-value="false"
  @change="(val) => toggleFieldVisibility(field.id, val)"
/>
```

#### 过滤可见字段
```typescript
const visibleFields = computed(() => {
  return props.fields.filter(field => field.isVisible !== false);
});
```

#### 导出时过滤
```typescript
const exportData = records.map(record => {
  const filteredValues = {};
  visibleFields.forEach(field => {
    filteredValues[field.name] = record.values[field.id];
  });
  return filteredValues;
});
```
