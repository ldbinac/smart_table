# 首页创建多维表格功能入口添加计划

## 问题分析

当前首页 (Home.vue) 存在以下问题：
1. **缺少创建多维表格的入口** - 页面上没有明显的"创建"按钮或操作选项
2. **空状态无引导** - 当没有 Base 时，页面显示为空白，没有引导用户创建
3. **操作单一** - 只有"设置"按钮，核心功能入口缺失

## 当前首页结构

```
Home.vue
├── header (home-header)
│   ├── h1: "Smart Table" (标题)
│   └── el-button: "设置" (跳转到设置页)
└── main (home-content)
    └── base-list (网格布局)
        └── el-card (base卡片列表) - 可点击跳转
```

## 解决方案

### 1. 在 Header 区域添加"创建多维表格"按钮

**位置**: 标题右侧，"设置"按钮旁边

**实现方式**:
- 添加一个醒目的主按钮 (el-button type="primary")
- 使用 Plus 图标 (已全局注册)
- 按钮文字: "创建多维表格" 或 "新建多维表格"
- 点击后弹出创建对话框

### 2. 添加创建多维表格对话框

**对话框内容**:
- 多维表格名称 (必填)
- 描述 (可选)
- 图标选择 (可选，使用 emoji 或图标选择器)
- 颜色选择 (可选，用于卡片主题色)

### 3. 在 Base 列表中添加"创建新 Base"卡片

**位置**: Base 列表的第一个位置

**样式**:
- 使用虚线边框或浅色背景区分
- 显示 Plus 图标和"创建多维表格"文字
- 点击后同样弹出创建对话框

### 4. 添加空状态引导

**当 bases 数组为空时显示**:
- 使用 Element Plus 的 el-empty 组件
- 显示"暂无多维表格"提示
- 提供明显的"创建多维表格"按钮

## 具体实施步骤

### 步骤 1: 修改 Home.vue 模板结构

1. 在 header 区域添加"创建多维表格"按钮
2. 在 base-list 中添加"创建卡片"作为第一个元素
3. 添加空状态显示 (v-if="baseStore.bases.length === 0")
4. 添加创建对话框 (el-dialog)

### 步骤 2: 添加响应式数据和方法

1. 添加 `createDialogVisible` 控制对话框显示
2. 添加 `createForm` 存储表单数据
3. 添加 `handleCreateBase` 方法调用 store 创建 Base
4. 添加 `openCreateDialog` 和 `closeCreateDialog` 方法

### 步骤 3: 添加样式

1. 创建卡片样式 (虚线边框、居中内容)
2. 空状态样式
3. 对话框表单样式

## 预期效果

1. 用户在首页可以直观地看到"创建多维表格"按钮
2. 无论是否有现有 Base，都能轻松找到创建入口
3. 创建流程简单明了，只需填写名称即可创建
4. 创建成功后自动刷新列表并显示新 Base
