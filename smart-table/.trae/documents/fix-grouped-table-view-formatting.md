# 分组表格视图字段显示格式修复计划

## 问题分析

当前 GroupedTableView 组件存在以下问题：

1. **单选/多选字段显示问题**：直接显示 ID 值而非名称，且缺少视觉样式（颜色标签）
2. **日期字段格式问题**：未遵循字段配置的时间显示规则（showTime）
3. **复选框样式问题**：仅显示 "✓" 符号，与标准表格视图的样式不一致
4. **数据行功能缺失**：
   - 缺少序号列
   - 缺少悬停复选按钮
   - 缺少行选择功能

## 修复方案

### 任务 1：修复字段显示格式

#### 1.1 单选/多选字段显示
- 修改 `getCellDisplayValue` 函数，根据字段 options 将 ID 转换为名称
- 添加颜色标签显示（参考 TableCell.vue 中的实现）
- 创建 SelectTag 组件或内联样式显示带颜色的选项

#### 1.2 日期字段格式
- 修改 `getCellDisplayValue` 函数，根据字段 options 中的 `showTime` 配置决定显示格式
- 支持 `YYYY-MM-DD` 和 `YYYY-MM-DD HH:mm:ss` 两种格式

#### 1.3 复选框样式
- 修改复选框显示方式，使用与 TableCell.vue 一致的样式
- 显示为带边框的勾选标记或自定义样式

### 任务 2：添加数据行功能

#### 2.1 添加序号列
- 在表头添加序号列（第一列）
- 在数据行显示行号（从 1 开始计数）
- 考虑分组内的行号还是全局行号

#### 2.2 添加悬停复选按钮
- 添加行选择状态管理（selectedRows）
- 鼠标悬停时显示复选框
- 支持点击复选框选择/取消选择行
- 支持 Ctrl/Cmd 多选、Shift 连选

#### 2.3 添加行选择功能
- 添加行点击选择功能
- 添加选中行样式（高亮显示）
- 添加全选/取消全选功能
- 向父组件 emit 选择事件

### 任务 3：样式统一

- 统一数据行样式与 TableView 一致
- 统一单元格边框、padding 等样式
- 统一悬停、选中状态样式

### 任务 4：测试验证

- 测试不同字段类型的显示
- 测试不同分组层级的显示
- 测试行选择功能
- 测试样式一致性

## 详细实现步骤

### 步骤 1：创建辅助组件

创建 `SelectTag.vue` 组件用于显示带颜色的单选/多选值：
- Props: option { id, name, color }
- 显示圆角标签，带背景色

### 步骤 2：重构 GroupedTableView

修改内容：
1. 添加 `selectedRows` 响应式状态
2. 添加 `rowNumber` 计算属性或方法
3. 重构 `getCellDisplayValue` 函数
4. 添加 `renderCell` 函数用于渲染不同类型的单元格
5. 添加行选择相关方法
6. 修改模板添加序号列和复选框

### 步骤 3：样式调整

- 添加序号列样式
- 添加复选框样式
- 调整选中行样式
- 统一单元格样式

### 步骤 4：集成测试

- 在 Base.vue 中测试分组视图
- 验证字段显示正确性
- 验证行选择功能
- 验证样式一致性
